Inregistrare model Employee in Django Admin

PASUL 1: In fisierul models.py din aplicatia dorita, se defineste modelul Employee.
Exemplu:
class Employee(models.Model):
    first_name = models.CharField(max_length=100)
    last_name = models.CharField(max_length=100)
    email = models.EmailField()

PASUL 2: In fisierul admin.py din aceeasi aplicatie, se importa modelul si se inregistreaza.
Exemplu:
from django.contrib import admin
from .models import Employee

admin.site.register(Employee)

PASUL 3: Asigura-te ca aplicatia este inclusa in INSTALLED_APPS din settings.py.

PASUL 4: Acceseaza interfata Django Admin si modelul Employee va aparea in lista pentru administrare.




Ce este un superuser si cum se creaza in Django

Un superuser este un utilizator cu toate permisiunile in aplicatia Django.
Are acces complet la interfata de administrare (Django Admin) si poate vizualiza, edita sau sterge orice model inregistrat.

Cum se creaza un superuser:

PASUL 1: In terminal, navigheaza in directorul proiectului Django.

PASUL 2: Ruleaza comanda:
python manage.py createsuperuser

PASUL 3: Introdu urmatoarele informatii cerute:
- Username
- Email (optional)
- Parola (de doua ori)

PASUL 4: Dupa creare, te poti autentifica cu acest superuser in pagina Django Admin la:
http://localhost:8000/admin/

Superuserul este folosit in principal pentru gestionarea aplicatiei in faza de dezvoltare sau pentru administrarea aplicatiei in productie (cu masuri de securitate corespunzatoare).
